# Module_Loader
模块加载器
-[Docs](https://github.com/trueFeeling/Module_Loader/wiki)
## usage
生产模式(未压缩)
```bash
# git clone https://github.com/trueFeeling/Module_Loader.git
# cd Module_Loader
# npm install
# npm run build
```
开发模式(已压缩)
```bash
# git clone https://github.com/trueFeeling/Module_Loader.git
# cd Module_Loader
# npm install
# npm run dev
```
